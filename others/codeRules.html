# 命名规则

  • 文件夹命名采用 camel 规则
  • 一般文件命名采用 camel 规则
  • 一般变量命名采用 camel 规则/下划线格式（由后端 传参/返回参数 决定）
  • 组件命名采用 pascal 规则
  • 常量命名采用全大写，下划线分割规则
  • 静态资源(图片等)命名采用 kebab-case 规则
